import telebot import json import threading import time from telebot import types # 1. Твій токен TOKEN = '8640349178:AAFnf9a5Evsyy00vnuL5Yld7zwSg5clmdIo' bot = telebot.TeleBot(TOKEN) # 2. Твій Telegram ID (Бот надсилатиме результати саме сюди) YOUR_USER_ID = 8708532759 # --- ФУНКЦІЇ ДЛЯ РОБОТИ З JSON --- def load_data(): """Завантажує дані з файлу""" try: with open('users.json', 'r', encoding='utf-8') as f: return json.load(f) except: return {"pending_action": None, "last_result": None} def save_data(data): """Зберігає дані у файл""" try: with open('users.json', 'w', encoding='utf-8') as f: json.dump(data, f, ensure_ascii=False, indent=4) except Exception as e: print(f"Помилка запису у файл: {e}") def set_agent_command(command): """Записує нову команду для агента на ПК""" data = load_data() data["pending_action"] = command data["last_result"] = None # Очищаємо старий результат save_data(data) # --- ФОНОВА ПЕРЕВІРКА РЕЗУЛЬТАТІВ --- def check_for_results(): """Стежить за файлом і надсилає відповіді від ПК""" while True: try: data = load_data() if data.get("last_result"): result_text = data["last_result"] bot.send_message(YOUR_USER_ID, f"📢 Відповідь від BAS:\n\n{result_text}") # Очищаємо результат, щоб не надсилати двічі data["last_result"] = None save_data(data) except Exception as e: print(f"Помилка фонової перевірки: {e}") time.sleep(1) # Перевірка кожну секунду # --- ОБРОБНИКИ КОМАНД --- @bot.message_handler(commands=['start']) def welcome(message): bot.reply_to(message, f"Вітаю! Твій ID: {message.from_user.id}\nНапиши 'меню', щоб відкрити панель.") @bot.message_handler(commands=['myid']) def my_id(message): bot.reply_to(message, f"Твій ID: {message.from_user.id}") @bot.message_handler(func=lambda m: m.text.lower() == "меню") def show_menu(message): markup = types.InlineKeyboardMarkup() # Твоє посилання на GUI web_app = types.WebAppInfo(url="https://deras23ua.github.io/my-bot-GUI/Gui.html") button = types.InlineKeyboardButton(text="Відкрити Помічник 🚀", web_app=web_app) markup.add(button) bot.send_message(message.chat.id, "Панель керування BAS:", reply_markup=markup) @bot.message_handler(content_types=['web_app_data']) def get_web_app_data(message): """Обробляє натискання кнопок у WebApp (якщо fetch не спрацював)""" try: web_data = json.loads(message.web_app_data.data) action = web_data.get('action') if action: set_agent_command(action) bot.send_message(message.chat.id, f"⌛ Запит '{action}' надіслано на ПК. Очікуйте відповідь...") except Exception as e: bot.send_message(message.chat.id, f"Помилка кнопки: {e}") # --- ГОЛОВНИЙ ЗАПУСК --- if __name__ == '__main__': # Запускаємо фонову перевірку в окремому потоці threading.Thread(target=check_for_results, daemon=True).start() print("Бот запущений і стежить за файлом users.json...") bot.polling(none_stop=True)
